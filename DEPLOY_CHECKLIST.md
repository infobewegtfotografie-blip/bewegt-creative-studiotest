# Deploy Checklist

1. Unzip this release package.
2. Replace the current GitHub repository files with the contents of this folder.
3. Commit with a clear message, for example:
   `Release BEWEGT Creative Studio v1.0.0 production`
4. Push to GitHub.
5. Wait for Netlify to redeploy.
6. Test these URLs:
   - https://bewegtcreative.com/
   - https://bewegtcreative.com/photography.html
   - https://bewegtcreative.com/video-production.html
   - https://bewegtcreative.com/live-streaming.html
   - https://bewegtcreative.com/graphic-design.html
   - https://bewegtcreative.com/podcast-production.html
   - https://bewegtcreative.com/thank-you
7. Test the contact form.
8. Test language switching EN / FR / DE.
9. Test the German service block and confirm `Videoproduktion` is fully visible.
10. In Google Analytics Realtime, confirm page views appear after deployment.
11. In Microsoft Clarity, confirm sessions start appearing after traffic.
